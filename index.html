📘 Dashboard Help

ChatGPT Apps — Category × Capability Dashboard

Overview

This dashboard provides a structured, explorable view of ChatGPT-connected applications, organized by categories and capabilities.

  • Categories (e.g. Design, Productivity, Business)
  • Capabilities (e.g. Interactive(API), Writes(API), File Search(MCP), Sync)

It helps users quickly answer questions like:

  • Which apps support Interactive APIs?
  • What design tools support file search?
  • Which apps match a specific capability combination?

The dashboard supports two complementary views:

  • Matrix View — high-level comparison
  • List View — detailed, filterable results

Data Model

  • App name
  • Category (single)
  • Capabilities (one or more)

Matrix View (Overview)

Purpose: Visual comparison across categories and capabilities.

  • Columns → Categories
  • Rows → Capabilities
  • Cells → Apps matching category and capability
  • Heatmap shading → Darker cells indicate more apps

Interactions

  • Click a cell to see apps inside it
  • Filter visible rows or columns
  • Export visible apps as CSV

List View (Search & Discovery)

Purpose: Precise filtering and app discovery.

  • Scrollable list of apps
  • Shows app name, category, and capability chips

Capability Query Builder

Filter apps using natural language or manual selectors.

Examples:

  • interactive
  • interactive and writes
  • sync but no file search
  • interactive api but no writes

The parser is case-insensitive and ignores spaces and punctuation (interactive(api) = interactive api).

Apply Button

  • Parses the text query (if present)
  • Applies the resulting filters

Clear Button

  • Resets text query
  • Clears include/exclude capabilities
  • Resets list results

Include Capabilities (AND)

Apps must have all selected capabilities.

Exclude Capabilities

Apps must have none of the selected capabilities.

Result Controls

  • Result count shows matching apps
  • Search within results filters by name, category, or notes

Export

Export visible apps (CSV) based on current filters and view.

Typical Use Cases

  • 🔍 Find compatible apps
  • 🧭 Explore categories
  • 📊 Gap analysis
  • 📁 Reporting

Tips & Best Practices

  • Use Matrix view for discovery
  • Use List view for precision
  • Combine Include + Exclude filters
  • Export after filtering

Known Limitations

  • Categories are single-valued per app
  • Capability matching is exact
  • Free-text search does not override capability filters
